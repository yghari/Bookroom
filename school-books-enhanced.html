🛠 Setup Instructions
  1. Open Google Sheets → create a new spreadsheet
  2. Click Extensions → Apps Script
  3. Paste the Apps Script code shown on the right into Code.gs
  4. Click Deploy → New Deployment → choose Web app
  5. Set Execute as: Me and Who has access: Anyone
  6. Copy the Web App URL and paste it above
  7. Click 🔄 Sync Now — your data pushes to Sheets!
